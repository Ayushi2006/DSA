## 🧠 Singly Linked List

This is a custom implementation of a singly linked list without using built-in data structures.

### ✨ Features

- Create a linked list
- Insert elements:
  - At the beginning
  - At the end
  - After a specific element
- Delete elements:
  -Delete first
  -Delete last
  -Delete a specific element
- Search
- Display the list
